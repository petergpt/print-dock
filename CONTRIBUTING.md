# Contributing

Thanks for your interest in Print Dock!

## Ways to contribute
- Report bugs or request features via GitHub issues
- Improve documentation
- Submit pull requests

## Development

```
./scripts/verify.sh
```

If you don’t have Xcode installed, tests will be skipped.

## Pull requests
- Keep changes focused and well‑scoped
- Include tests for behavioral changes when possible
- Update docs if the public surface changes
